How It Works
Three steps to multi-cloud.
No Kubernetes expertise needed. Just push code.
01 📁
Fork & Drop Your App
Fork the GitHub repo. Drop your app's Kubernetes manifest (or use the template) into manifests/your-app/. That's it.
02 🔗
Connect to ArgoCD
Link your forked repo to the ArgoCD instance. ArgoCD watches your repo 24/7 and auto-syncs any changes directly to the cluster.
03 🚀
Push. It Deploys.
Every git push to main triggers an automatic rollout across AWS, GCP, and bare-metal nodes simultaneously.
